# 英雄聯盟抽籤工具

靜態版英雄聯盟位置抽籤工具，可直接透過瀏覽器使用。

- 純前端 HTML/CSS/JavaScript
- 歷史數據儲存在使用者自己的瀏覽器 localStorage
- 不需要登入、後端 API 或資料庫
- 適合 GitHub Pages 或其他靜態網站託管

## 使用方式

開啟頁面後輸入玩家名稱，加入名單後即可抽籤。抽籤結果會顯示每位玩家的主要位置與次要位置，並可複製結果。

## 資料儲存說明

歷史數據只會存在目前瀏覽器與目前網站網址底下，不會上傳到伺服器，也不會在不同裝置或不同網址之間自動同步。

例如以下來源會各自擁有獨立資料：

- 本機檔案
- 本機 LAN server
- GitHub Pages 網址
- 不同手機或不同瀏覽器

如果清除瀏覽器資料，歷史數據也可能一併消失。
